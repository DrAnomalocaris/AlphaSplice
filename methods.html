Methods

The protein foldings were run using local collabfold. on my laptop (using CPU) with relaxation.

The entire analysis is performed using a Snakefile.

The snakefile is designed to arbitrarily work with any gene/transcript by using ENSG codes, although it is designed to work only for humans, adapting it for any organism should not be too complicated.
